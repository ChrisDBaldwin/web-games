This is the repo of html games that Chris makes when he thinks of them. Some of them will make it to the subdomain, games.voidtalker.com

## Games

- **planning-joker.html** — A card game with a planning twist
- **snowball.html** — Snowball game
- **motion-storm.html** — Webcam-powered physics game. Balls rain from the sky and your body movements create force fields that push them around. Guide balls into golden score rings, keep them off the floor, and survive escalating waves. Requires a webcam and HTTP server (`python3 -m http.server 8000`).
- **ram-butt.html** — A two-player game of stubbornness. Ram your opponent off the field in head-to-head local multiplayer.
- **knot-equivalence.html** — 3D WebGL knot identification game. Two knots are shown side by side in random orientations — guess whether they're the same type. Features 8 knot types, drag-to-rotate 3D view, and a 2D textbook-style diagram mode with proper over/under crossings.
- **dream-logic.html** — Endless fog-of-war puzzle roguelike set inside a sleeping mind. Explore outward through procedurally generated dream grids, collect memories, and solve puzzles. When the dreamer wakes, the world collapses from the edges — race back to the center before it dissolves. Each night the dream goes deeper, harder, and stranger.
- **clown-truck.html** — Arcade-style clown deployment game. Drive a clown truck around a procedural city and dispatch your 6-clown crew using fighting-game button layouts (U/I/O + J/K/L). Each clown has its own AI personality — they autonomously chase down dudes and collect trash while you manage positioning and timing. Even the driver can be deployed.
- **railgrind.html** — Infinite procedurally generated mine-cart runner with a cel-shaded, graffiti-tagged Jet Set Radio aesthetic. One button: jump. Clear gaps and spiky rail-sitters, chase distance, die hard. Coyote time and jump buffering for a jump that feels right.
