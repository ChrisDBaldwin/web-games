This is the repo of html games that Chris makes when he thinks of them. Some of them will make it to the subdomain, games.voidtalker.com

## Games

- **planning-joker.html** — A card game with a planning twist
- **snowball.html** — Snowball game
- **motion-storm.html** — Webcam-powered physics game. Balls rain from the sky and your body movements create force fields that push them around. Guide balls into golden score rings, keep them off the floor, and survive escalating waves. Requires a webcam and HTTP server (`python3 -m http.server 8000`).
- **ram-butt.html** — A two-player game of stubbornness. Ram your opponent off the field in head-to-head local multiplayer.
